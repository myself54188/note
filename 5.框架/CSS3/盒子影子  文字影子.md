# **盒子影子  文字影子**

盒子阴影：

​		可以用 box-shadow 属性为盒子添加阴影。

```css
语法：
box-shadow: h-shadow v-shadow blur spread color inset;
```

![](https://github.com/myself54188/picx-images-hosting/raw/master/盒子阴影.8ojmbsmy8s.webp)



文字阴影:

```css
语法：
text-shadow: h-shadow  v-shadow  blur  color;
```
